# Yauncai's Bistro

A developer-themed restaurant page built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## About

This project is a practice exercise focused on **DOM manipulation** and **dynamic content rendering** using vanilla JavaScript. No HTML is written directly for the page content — everything is created and injected via JS modules.

## What I Practised

- Creating and appending DOM elements with JavaScript
- Organising code into ES6 modules
- Tabbed navigation without a page reload
- Bundling assets with Webpack
- Deploying a Webpack project to GitHub Pages

## Tech Stack

- HTML / CSS
- Vanilla JavaScript (ES6 modules)
- Webpack 5
